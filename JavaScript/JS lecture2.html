JavaScript Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations like addition, subtraction, multiplication, and division.

Example: let sum = 5 + 3; → Result is 8

Logical Operators

Logical operators combine or invert conditions. Common ones are AND (&&), OR (||), and NOT (!).

Example: (5 > 3 && 10 > 7) → Result is true

String Concatenation

String concatenation joins two or more text strings using the + operator.

Example: "Hello" + " " + "World" → Result is "Hello World"
